Install

Get started in seconds.

Same CLI everywhere. Choose the path that matches your machine.

Any distro · macOS · BSD

One-liner (non-interactive)

curl -fsSL https://raw.githubusercontent.com/gpteamofficial/vopk/main/src/installscript.sh | sudo bash -s -- -y

Works everywhere (Linux/macOS/BSD). The script auto-detects your package manager and installs needed bits. On Arch we recommend the AUR package below, but this one-liner also works.

Linux

Debian / Ubuntu / Fedora / openSUSE / Alpine / Void / Gentoo

curl -fsSL https://raw.githubusercontent.com/gpteamofficial/vopk/main/src/installscript.sh | sudo bash -s -- -y

Same installer everywhere; auto-detects apt/pacman/dnf/yum/zypper/apk/xbps/emerge and handles curl if missing.

Arch / AUR

Official AUR (recommended)

# install yay if needed
sudo pacman -S --needed base-devel git yay

# then install vopk from AUR
yay -S vopk

AUR package name: vopk. The universal installer also works on Arch, but AUR is the preferred path.

Fedora / RHEL

dnf / yum

curl -fsSL https://raw.githubusercontent.com/gpteamofficial/vopk/main/src/installscript.sh | sudo bash -s -- -y

Installer will handle curl installation and use dnf/yum automatically.

macOS

Script install

curl -fsSL https://raw.githubusercontent.com/gpteamofficial/vopk/main/src/installscript.sh | sudo bash -s -- -y

Installer uses Homebrew under the hood if available; no manual steps needed.

BSD

FreeBSD / NetBSD / OpenBSD

curl -fsSL https://raw.githubusercontent.com/gpteamofficial/vopk/main/src/installscript.sh | sudo bash -s -- -y

Not in BSD repos; the universal installer handles pkg/pkgin/pkg_add and curl setup.

Containers & fallback

dpkg-only & vmpkg

vopk install ./package.deb
vopk vm install example

Survives minimal images; bridges env flags into vmpkg when used.

Why VOPK

Unified, predictable, and a bit opinionated.

One mental model

update, upgrade, install, remove, purge, autoremove, search, list, show, clean — identical verbs everywhere.

Safety-first UX

Color-coded logs, explicit confirmations, dry-run preview without prompts, and distro-aware name mapping.

Script ready

Short aliases (`i`, `rm`, `up`, `fu`, `ls`, `s`, `si`) plus raw passthrough via `vopk script-v` when you need it.

Backends that matter

apt/apt-get, pacman+yay, dnf/yum, zypper, apk, xbps, emerge, Homebrew, pkg/pkg_add/pkgin, vmpkg.

Helpers built-in

doctor, sys-info, dev-kit installer, fix-dns, disk/mem/top/ip helpers — reduce context switching.

Respectful defaults

No hidden sudo escalation; honors VOPK_* env vars, APKG legacy flags, and your color/quiet preferences.

Commands

Memorable verbs, sensible aliases.

update

Refresh package metadata.

upgrade / fu

Upgrade packages (full upgrade via `full-upgrade`/`fu`).

install / i

Install packages with smart name mapping per distro.

remove / purge

Remove packages (purge configs where supported).

autoremove

Clean orphaned deps (pacman/dnf/pkg/brew aware).

search / list / show

Consistent discovery helpers.

clean

Cache cleanup with dry-run previews.

script-v

Direct passthrough to the underlying backend for scripts.
